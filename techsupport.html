Услуги отдела технической поддержки

Помогаем по любым вопросоам связанным с разработкой, внедрением и сопровождений технических процессов.

  • 01

    Подбор продукта

    Подбор продукта осуществляем с учетом особенностей технологий, материалов и оборудования заказчика. В ассортименте компании: полимерные дисперсии, промышленные клеи для мебели, деревообработки и упаковки, клеи для укладки напольных покрытий, строительная химия.

  • 02

    Выезд на предприятие

    Опционно служба поддержки выезжает на предприятии заказчика. Технологи знакомятся с производственным процессов, проводят консультацию по его наладке, определяют причины некорректной работы.

  • 03

    Выбор решения

    По результатам полученных данных предлагаем подходящее решение: продукт из ассортимента, доработка рецептуры под технические особенности или разработка с нуля под специфику процесса.

  • 04

    Доработка продукта

    Продукт из ассортимента «Хома» по необходимости дорабатывается под технологические особенности предприятия заказчика. По результатам тестирования технолог компании определит, какие параметры необходимо изменить.

  • 05

    Разработка с нуля

    Собственный научный ресурс компании позволяет разрабатывать продукты с индивидуальными свойствами. Продукты «Хома» могут быть созданы с нуля под специфику процесса производства. Каждый этап разработки под контролем - ученые управляют свойствами и характеристиками будущих продуктов.

  • 06

    Внедрение

    Служба поддержки компании помогает внедрить продукт на производстве. Технологи проводят тестирования, настраивают техпроцесс, доводят испытания до положительного результата.

  • 07

    Сопровождение

    Клиент компании сопровождается на всем пути использования продукта «Хома». Специалисты компании проводят оперативные консультации, по необходимости приезжают на предприятие для устранения неполадок техпроцессов и решения других вопросов.
